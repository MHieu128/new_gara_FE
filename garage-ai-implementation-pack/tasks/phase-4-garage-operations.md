# Phase 4 — Garage Operations Tasks

## Goal
Build the internal garage workflow after a customer arrives.

## Tasks

### Job Cards
- [ ] Create job card from booking.
- [ ] Add customer complaint.
- [ ] Add odometer.
- [ ] Add fuel level.
- [ ] Add internal notes.
- [ ] Update job card status.

### Inspection
- [ ] Upload vehicle images.
- [ ] Categorize image type.
- [ ] Add notes to image.
- [ ] View inspection history.

### Technician Tasks
- [ ] Add tasks.
- [ ] Assign technician.
- [ ] Start task.
- [ ] Complete task.
- [ ] Track estimated and actual time.

### Inventory
- [ ] Track inventory by branch.
- [ ] Add material usage to job card.
- [ ] Decrease stock on usage.
- [ ] Alert when below reorder threshold.

### Warranty
- [ ] Issue warranty after job completion.
- [ ] Generate warranty code.
- [ ] View warranty by vehicle/customer.
- [ ] Mark warranty claimed/voided.

## Definition of Done
- A booking can become a job card.
- A job card can be completed.
- Materials can be tracked.
- Warranty can be issued.
