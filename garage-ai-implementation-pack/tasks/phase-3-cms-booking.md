# Phase 3 — CMS, Lead Management and Booking Tasks

## Goal
Create operational features for managing leads, bookings and quotations.

## Tasks

### CRM
- [ ] Lead list with filters.
- [ ] Lead detail.
- [ ] Lead status workflow.
- [ ] Lead assignment.
- [ ] Customer creation/update.
- [ ] Vehicle creation/update.

### Booking
- [ ] Create booking.
- [ ] Update booking status.
- [ ] Filter bookings by branch/date/status.
- [ ] Check booking overlap.
- [ ] Assign employee.
- [ ] Convert lead to booking.

### Quotation
- [ ] Create quotation.
- [ ] Add quotation items.
- [ ] Calculate subtotal/discount/total.
- [ ] Send quotation status.
- [ ] Accept/reject quotation.

### Notifications
- [ ] Add notification queue abstraction.
- [ ] Notify staff when new lead is created.
- [ ] Notify customer when booking is confirmed.
- [ ] Log notification failures.

## Definition of Done
- Staff can manage lead lifecycle.
- Staff can schedule a booking.
- Staff can create a quotation.
- System validates pricing from database.
