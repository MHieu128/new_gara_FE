# Prompt — Initialize Vue Static Site

Implement the initial Vue 3 static website foundation.

## Requirements
- Vue 3
- Vite
- TypeScript
- TailwindCSS
- Vue Router
- Path alias `@` to `src`

## Create
```text
src/app/App.vue
src/components/layout/AppHeader.vue
src/components/layout/AppFooter.vue
src/components/layout/MobileNavigation.vue
src/router/index.ts
src/views/HomeView.vue
src/views/ProductsView.vue
src/views/ProductDetailView.vue
src/views/ServicesView.vue
src/views/ServiceDetailView.vue
src/views/BranchesView.vue
src/views/ArticlesView.vue
src/views/ArticleDetailView.vue
src/views/ContactView.vue
src/main.ts
src/style.css
```

## Routes
```text
/
/san-pham
/san-pham/:slug
/dich-vu
/dich-vu/:slug
/chi-nhanh
/tin-tuc
/tin-tuc/:slug
/lien-he
```

## UI Direction
Create a premium automotive website look with dark hero, strong CTA buttons, clean navigation, responsive mobile menu and professional typography.

## Do Not
- Do not add backend.
- Do not add database.
- Do not add heavy UI libraries.
