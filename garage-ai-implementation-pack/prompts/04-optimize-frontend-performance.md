# Prompt — Optimize Frontend Performance

Review and optimize the Vue 3 static site.

## Targets
- Lighthouse Performance 90+
- Accessibility 90+
- Best Practices 90+
- SEO 90+

## Check
- Route-level code splitting.
- Image lazy loading.
- WebP/AVIF usage.
- No heavy dependencies.
- No direct JSON imports in components.
- No oversized hero images.
- No layout shift from images.
- No unnecessary watchers.
- No huge hidden DOM blocks.
- SEO meta support.
